# AI学習プラン提案アプリ

生成AIを活用し、入力した学習目標と使える時間から学習プランを提案するWebアプリです。

## 使用技術
- HTML / CSS / JavaScript
- Gemini API（生成AI）

## 動かし方
1. `index.html` 内の `YOUR_API_KEY_HERE`(411行目) を、ご自身の Gemini API キーに置き換えてください。
2. ブラウザ（または VS Code の Live Server）で `index.html` を開きます。
3. 学びたい内容と使える時間を入力し、ボタンを押すと学習プランが表示されます。

## セキュリティについて
API キーはセキュリティのため、ソースコードには含めていません。お手数ですが、ご自身のキーを設定してご利用ください。
